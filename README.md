# Instalación Dependecias Principales

```bash
bun add @nestjs/jwt @nestjs/passport @nestjs/cache-manager passport passport-custom passport-google-oauth20 passport-jwt passport-local @prisma/client class-transformer class-validator bcrypt speakeasy qrcode cookie-parser dotenv resend uuid cache-manager @keyv/redis
```

# Instalación Dependecias Desarrollo

```bash
bun add -D @types/bcrypt @types/cookie-parser @types/passport-google-oauth20 @types/passport-jwt @types/passport-local @types/qrcode @types/speakeasy @types/uuid prisma
```
